# TypeScript Concepts & Solutions

## Overview

This assignment covers essential TypeScript concepts and practical implementations, combining theoretical understanding with hands-on problem-solving.

## Contents

### 📚 Blog Articles

#### [Blog 1: TypeScript Utility Types - Pick and Omit](blog-1.md)
Explores how `Pick` and `Omit` utility types help create specialized "slices" of interfaces while maintaining DRY (Don't Repeat Yourself) principles. This article discusses:
- How `Pick` extracts specific properties from an interface
- How `Omit` removes specific properties from an interface
- Real-world examples to prevent code duplication

#### [Blog 2: Four Pillars of OOP](blog-2.md)
Explains the four fundamental pillars of Object-Oriented Programming and their role in managing complexity in large-scale TypeScript projects:
- **Inheritance**: Reusing code through class hierarchy
- **Polymorphism**: Implementing multiple forms of behavior
- **Abstraction**: Hiding implementation details
- **Encapsulation**: Bundling data and methods together

### 💻 Solutions

The [solutions.ts](solutions.ts) file contains TypeScript implementations addressing five key problems:

1. **Filter Even Numbers** - Array filtering with type safety
2. **Reverse a String** - String manipulation
3. **Type Guard for String or Number** - Union type handling with type guards
4. **Generic Function to Get Property Value** - Generic types with key constraints
5. **Intersection Types for Book and Read Status** - Combining multiple types for complex data structures

## Learning Objectives

- Understand and apply TypeScript utility types (`Pick`, `Omit`)
- Master OOP principles in TypeScript
- Work with generics, union types, and intersection types
- Implement type guards for better type safety
- Write clean, reusable, and maintainable TypeScript code
